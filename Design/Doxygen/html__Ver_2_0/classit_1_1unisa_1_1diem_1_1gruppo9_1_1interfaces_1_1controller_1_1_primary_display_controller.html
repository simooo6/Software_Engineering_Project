Constructs an instance of the PrimaryDisplayController.

This constructor initializes the ChangeView instance used for managing view transitions.

Invariant
The view object is declared as a ChangeView.
Precondition
The ChangeView class must be available in the project and correctly defined.
Postcondition
A new instance of PrimaryDisplayController is created with a valid ChangeView object.

Handles the event triggered when the "Add Contact" button is pressed.

This method performs the following actions:

  • Loads the contact list using the ContactManager#loadContact() method.
  • Switches the scene to AddView.fxml using the ChangeView#addContactView(ActionEvent, ContactManager) method.
