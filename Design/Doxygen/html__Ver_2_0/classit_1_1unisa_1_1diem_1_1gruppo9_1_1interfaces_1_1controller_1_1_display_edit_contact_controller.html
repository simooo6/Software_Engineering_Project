◆ initialize()

void it.unisa.diem.gruppo9.interfaces.controller.DisplayEditContactController.initialize ( )

Initializes the controller after its elements have been loaded.

This method sets up the logic for disabling the "editButton" if both the name and surname fields are empty. It uses a property binding to monitor changes in the text fields.

Invariant
Texfields phoneNumber1,phoneNumber2,phoneNumber3 must only contain numbers or are empty.
Precondition
The FXML elements must be properly loaded and connected.
Postcondition
The "editButton" will be disabled if both the "nameField" and "surnameField" are empty.
