The "BricoRubrica" is an AddressBook-like software that provides multiple data structures in order to memorize and operate on a list of contacts via GUI, as well as an explictly-user-activated mechanism that enables to overwrite a file with all the information contained in the address book. Each Contact is identified by a set of informations and each Contact is contained in a ContactManager object.
