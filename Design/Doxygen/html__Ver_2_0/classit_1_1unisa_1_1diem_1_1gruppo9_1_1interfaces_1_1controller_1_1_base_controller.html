void it.unisa.diem.gruppo9.interfaces.controller.BaseController.setContacts ( ContactManager contacts)

Sets the ContactManager instance for the controller.

This method implements the ContactManagerAware#setContacts(it.unisa.diem.gruppo9.logic.ContactManager) interface. It allows dynamic injection of a ContactManager instance, which is stored in the contacts field. This instance can then be utilized by subclasses to perform contact management operations, ensuring a consistent interaction with the ContactManager.

This method also represents the public contract between the interface logic and the controller layer, facilitating communication between an instance of ChangeView and the system controllers like DisplayEditContactController, DisplayContactsController, and DisplayAddContactController.
