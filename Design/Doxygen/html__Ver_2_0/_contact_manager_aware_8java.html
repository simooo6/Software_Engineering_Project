interface  it.unisa.diem.gruppo9.logic.ContactManagerAware
 

Detailed Description

Represents an interface to define a contract for classes that require a ContactManager instance.

Classes implementing the ContactManagerAware interface, such as BaseController in the "it.unisa.diem.gruppo9.interfaces.controller" package, are expected to provide a concrete implementation of the setContacts(ContactManager contacts) and getContacts() methods or to extend a class that implements this very interface. This methods are intended to associate a ContactManager instance with the implementing class, enabling interaction with the contact management logic.

The primary purpose of this interface is to establish a standardized way for classes in the application to access and utilize the functionality of the ContactManager class or to gain access to the associated ContactManager. This ensures that classes implementing this interface can access the ContactManager instance after it has been set. This two-way interaction enhances flexibility and modularity within the application architecture.

The relationship between this interface and the ContactManager class is a "use" relationship, meaning the interface depends on ContactManager for its functionality.
