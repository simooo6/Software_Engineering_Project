◆ initialize()

void it.unisa.diem.gruppo9.interfaces.controller.DisplayContactsController.initialize ( )

Initializes the controller class. It is called automatically after the FXML file is loaded.

Specifically it initialize the table view with a list of contact retrived from the ContactManager object; configure the table view column with their respective values, and initialize the binding property of the button: search, cancel, edit and delete.

Invariant
the ContactManager instance must be not modified
Precondition
The tableView and the buttons must be correctly associated with their respective references in the controller
the getContact() method must return a valid ContactManager object
The contact object retured by getListOfContacts() must contains consistent values
Postcondition
The table view must be correctly populated with the contacts
Contact's data must be correctly displayed
The button are enabled/disabled dinamically based on the binding conditions
