int it.unisa.diem.gruppo9.logic.NameAndSurnameComparator.compare ( Contact o1,
Contact o2 )

Compares two Contact objects by their name and surname.

This method compares the name attributes of the two contacts first. If the names are identical, it proceeds to compare their surname attributes. The comparison follows the natural lexicographic ordering of String.

Precondition
A NameAndSurnameComparator instance must be declared with the associated default constructor
name != null || surname != null
Both o1 and o2 must not be null (the contacts must be valid).
Postcondition
The method does not modify the state of the o1 or o2 objects.
If both contacts have a non-empty surname, the comparison is first made on the surname, and then on the name if the surnames are identical.
If the surname is empty for one contact, let be it the first, the comparison is made between the name of the first and the surname of the second and viceversa.
If the surname is empty for both contacts, the comparison is made based on the name.
If both the surname and name are identical the last added contact is the last visualized while keeping the sorted list valid, so that two identical contacts are still sorted in comparison to the others.
The Contact instances must not be modified in their content
Parameters
o1the first Contact object to be compared.
o2the second Contact object to be compared.
Returns
a negative integer, zero, or a positive integer as the first contact is less than, equal to, or greater than the second contact, respectively.
