Contact it.unisa.diem.gruppo9.logic.ContactAware.getSelectedContact ( )

Retrieves the Contact instance that has been selected in the graphical user interface.

This method provides access to the currently selected Contact, allowing the implementing class, and precisely its subclasses, to interact with the contact's data without directly managing its creation or selection logic.

Invariant
The object that is an instance of Contact must never be cast to another object.
The object that is an instance of Contact must never be modified in this method. It means that the name, surname, phoneNubers and emails that represents the contect of a contact must remain the same.
  • Precondition
    The Contact instance must have been initialized and set using the setSelectedContact(Contact contact) method.
    Returns
    the Contact instance currently selected in the interface.

void it.unisa.diem.gruppo9.logic.ContactAware.setSelectedContact ( Contact contact)

Sets the selected Contact instance for use in the implementing class.

This method allows the implementing class to associate a Contact instance that has been selected within the graphical user interface. The selected Contact can then be used for further operations, such as displaying or editing its details.

Invariant
