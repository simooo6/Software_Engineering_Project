◆ loadContact()

ContactManager it.unisa.diem.gruppo9.logic.ContactManager.loadContact ( ) throws FileNotFoundException, IOException

Loads the data structure from a defined file.

This method reads the contacts data from a CSV file named "rubrica.csv" and populates the contacts list. Each row in the file represents a contact, with fields separated by semicolons. If the file is empty, the ContactManager remains unchanged.

Invariant
The contacts list must not contain invalid or null Contact objects after the method execution.
The file "rubrica.csv" must follow the expected format: each row must contain exactly 8 fields, corresponding to surname, name, three phone numbers, and three emails.
Postcondition
The contacts list contains all valid Contact objects read from the file.
The method does not modify the existing data structure if the file is empty.
Returns
The ContactManager object with the updated contacts list.
Exceptions
FileNotFoundExceptionif the file "rubrica.csv" does not exist.
IOExceptionif an error occurs during file reading.

Sorts the ObservableList of contacts using the specified Comparator. Reorders the contacts in the internal ObservableList according to the order defined by the provided Comparator. By default, the NameAndSurnameComparator is used to sort the list by name and surname in lexicographic order.
