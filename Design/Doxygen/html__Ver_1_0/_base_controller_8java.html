Detailed Description

Provides a base implementation for setting a ContactManager instance in controllers.

The BaseController class implements the ContactManagerAware interface, allowing controllers that extend this class to have a ContactManager instance injected dynamically. This simplifies the management of contact data by ensuring that all extending controllers have access to an instance of ContactManager. Furthermore BaseController represents for a ChangeView instance the public contract with DisplayEditContactController, DisplayContactsController and DisplayAddContactController.

Responsibilities

  • Implements the setContacts(it.unisa.diem.gruppo9.logic.ContactManager) method from the ContactManagerAware interface to enable dependency injection of a ContactManager instance.
  • Provides a protected ContactManager field accessible to subclasses such as:
    • DisplayEditContactController
    • DisplayContactsController
    • DisplayAddContactController

Usage

The BaseController class is not abstract and can be extended by other controllers in the it.unisa.diem.gruppo9.interfaces.controller package. Subclasses automatically inherit the implementation of the setContacts(it.unisa.diem.gruppo9.logic.ContactManager) method and can access the protected contacts field to work with contact data.
