Detailed Description

Manages scene transitions in the application.

The ChangeView class is responsible for switching between different FXML scenes based on user interactions, such as button clicks in the GUI. It provides methods to manage transitions between specific views, enabling seamless navigation within the application.

Responsibilities

  • Handles scene changes triggered by button events in the application's GUI.
  • Works closely with controllers to manage navigation between views.
  • Ensures consistent transitions between the following FXML files:
    • ContactView.fxml
    • AddView.fxml
    • EditView.fxml

Usage

The ChangeView class is utilized by the following controllers in the it.unisa.diem.gruppo9.interfaces.controller package:

  • PrimaryDisplayController
  • DisplayContactsController
  • DisplayAddContactController
  • DisplayEditContactController

Each controller calls the appropriate method in ChangeView to handle the transition to the desired scene.

Scene Transitions

Each method in this class is responsible for transitioning to a specific view:

  • contactView: Called by all controllers listed above to navigate to ContactView.fxml.
  • addContactView: Called by all controllers listed above to navigate to AddView.fxml.
  • editContactView: Exclusively called by DisplayContactsController to navigate to EditView.fxml.
