void it.unisa.diem.gruppo9.interfaces.controller.BaseController.setContacts ( ContactManager contacts)

Sets the ContactManager instance for the controller.

This method is part of the ContactManagerAware interface implementation and allows subclasses to receive and manage a ContactManager instance dynamically. The injected ContactManager is stored in the contacts field, making it accessible to subclasses. In this sense the method represents the public contract between the interfaces.change section and the interfaces.controller section of the system. An instance of ChangeView is so allowed to use a ContactManager object and this very method to set it when an instance of DisplayEditContactController, DisplayContactsController or DisplayAddContactController needs it.

Invariant
ContactManagerAware#setContacts(it.unisa.diem.gruppo9.logic.ContactManager)
Precondition
The contacts parameter must not be null.
The contacts parameter must be a valid instance of ContactManager that has been properly initialized.
Postcondition
The contacts field of this instance is set to the provided contacts parameter.
The contacts field is accessible by subclasses of BaseController.
Parameters
contactsthe ContactManager instance to be associated with the controller.
