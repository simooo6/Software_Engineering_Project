Detailed Description

Represents a single contact in the address book.

The Contact class defines a set of parameters that characterize a contact. Each contact is represented as an instance of this class and can have mutable attributes, which can be modified using the provided setter methods.

A Contact consists of the following attributes:

  • name: the first name of the contact;
  • surname: the last name of the contact;
  • phoneNumber: an array of up to three phone numbers;
  • email: an array of up to three email addresses.
