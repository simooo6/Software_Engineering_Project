void it.unisa.diem.gruppo9.interfaces.change.ChangeView.addContactView ( ActionEvent event,
ContactManager contacts )

Transitions to the AddView.fxml scene.

This method switches the current scene to the AddView.fxml file. It is called by the following controllers when user interactions require navigation to the "Add Contact" view:

  • PrimaryDisplayController
  • DisplayContactsController
  • DisplayAddContactController
  • DisplayEditContactController
Parameters
eventthe action event that triggered the scene change.
contactsthe ContactManager instance used to manage contact data.
Invariant
The ContactManager instance must be unchanged during the transition
Precondition
event != null The action event triggering the scene change must not be null.
contacts != null The ContactManager must not be null.
Postcondition
The current scene is replaced with AddView.fxml.
